# event-ticket-booking-system

Event Ticket Booking System with QR Code Validation - A full-stack Java Spring Boot and React application for managing event tickets with secure QR code validation.

## Features
- User registration and authentication
- Event browsing and ticket booking
- QR code generation for tickets
- Admin dashboard for event management
- Seat selection system
- Ticket validation with QR scanning
- Modern responsive UI with colorful design

## Tech Stack
- **Backend**: Java Spring Boot, MySQL
- **Frontend**: React, TypeScript, CSS3
- **Authentication**: JWT tokens
- **QR Codes**: ZXing library

## Quick Start
1. Run `start-full-system.bat` to start both backend and frontend
2. Access the application at http://localhost:3000
3. Default admin: admin@event.com / admin123
